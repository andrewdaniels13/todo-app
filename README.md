# Todo App – TypeScript Demo

A simple command-line Todo app built with TypeScript. Created as a demo project for Mault onboarding.

## Setup

**1. Install dependencies**
```bash
npm install
```

**2. Run the app**
```bash
npm start
```

**3. Build to JavaScript**
```bash
npm run build
```

## Project Structure

```
my-toy-project/
├── index.ts        ← main app logic
├── tasklist.md     ← Mault task list
├── package.json    ← project config & scripts
├── tsconfig.json   ← TypeScript config
└── README.md       ← this file
```

## Features

- Add todos with validation
- Complete todos by ID
- Delete todos by ID
- List all todos with status

## Requirements

- Node.js v18+
- npm v8+
